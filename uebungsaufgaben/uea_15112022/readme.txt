Ziel: 
Du hast eine eigene Website z.B als Web-Visitenkarte


Vorgehen
1. Lege eine Verzeichnis mit deinem Namen an
2. Füge die (leeren) Verzeichnisse img und css an.
3. Lege im "Hauptverzeichnis" die Datei Index.html an. Die Datei Index.html stellt dein HTML Dokument dar und soll alle Inhalte enthalten.
4. Binde eine externe CSS Datei ein, über die du das Design deiner Seite steuerst.
5. Deine Website sollte über die Abschnitte header, content, Footer verfügen.

Inhalt der Seite
Die Seite sollte einen Lebenslauf enthalten, nutzen deinen eigenen oder einen ausgedachten.
Die Seite sollte folgende Informationen bieten:
- Wer ist die Person, Kurzbeschreibung als Fließtext
- Was waren die beruflichen Positionen,? Darstellung als Liste oder Tabelle
- Welche Fähigkeiten hat sie?
- Gibt es ein Bild?
- Wie kann ich mit der Person in Kontakt treten?
- Gibt es Social Media Kanäle?
Ansonsten... Lass dich im web inspirieren, sei kreativ.

Zur Info
Nicht vergessen, jede Seite benötigt ein Impressum. Siehe einen Link zum Impressum im Footer vor.
